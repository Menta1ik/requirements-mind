# Контекст Проекта: Example App

## 1. Vision и Суть проекта
Example App — это локальное приложение для управления персональными заметками в формате Markdown с поддержкой автосохранения и гибкого тегирования.

## 2. Ключевые заинтересованные стороны
* **Конечные пользователи:** Писатели, разработчики и студенты, которым нужен быстрый локальный markdown-редактор.
* **Разработчики:** Команда, реализующая приложение.

## 3. Бизнес-требования
* Обеспечить сохранность данных с помощью мгновенного автосохранения (локально на диск).
* Ускорить навигацию через систему тегов.

## 4. Нефункциональные требования
* Формат хранения: чистый Markdown.
* Локальный доступ: zero-cloud на первом этапе.
